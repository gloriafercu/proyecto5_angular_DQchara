# PROYECTO 5: D'Qchara

## 1. FRONTEND

D'Chara is developed with Angular 16 and Node 16.14.0

## Git clone

```
git clone https://github.com/gloriafercu/proyecto5_angular_DQchara.git
```

## Install

```
npm install
```

## Run API server

It works with json-server in https://localhost:5000

```
npm run serverAPI
```
